## Key Files Explained

### ğŸ”´ CRITICAL FILES (Affects Deployment)
- **index.html** - Main HTML file. Must have:
  - Correct favicon path: `href="/portfolio/vite.svg"`
  - Encoded Google Fonts URL: `&amp;` not `&`
  - Root div: `<div id="root"></div>`

- **public/404.html** - GitHub Pages SPA redirect. Must contain:
  - JavaScript that redirects 404s to root with path in sessionStorage

- **vite.config.js** - Build configuration. Must have:
  - `base: '/portfolio/'` for GitHub Pages

- **src/App.jsx** - Main component. Contains:
  - OnePagePortfolio (all sections on single page)
  - StickyNavbar (fixed navigation)
  - FloatingBubbles (background)
